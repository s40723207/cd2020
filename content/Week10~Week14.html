Week10

建立google api：

clone cd2020pj1倉儲

進入Google API，建立專案。

輸入專案cd2020pj1建立。

啟用API和服務，選Google+ Domains API。

再選擇內部，輸入名稱，設定憑證。

點選建立憑證及OAuth用戶端ID。

建立網路應用程式。

複製ID及密碼，命名為oauth_s學號.txt 存在tmp下。

接著設定來源及URL，接著存儲。

用leo打開cd2020pj1.leo。將路徑及名稱改為剛剛存的txt。

完成後存儲，按ROC。

在CMD輸入pip install authomatic。

Week11

Ubuntu設定

下載vitualbox、CoppeliaSim、ubuntu、四輪車控制器。

開啟vitualbox，匯入ubuntu。

開啟lxterminal，輸入xterm&開啟。

打開coppeliasim，輸入ls。

依序打cd coppeliasim4_rev4　ls　cd CoppeliaSim_Edu_V4_0_0_Ubuntu18_04　ls　./coppeliaSim.sh

打開檔案sdc.ttt。

開啟啟動器，並開啟四輪車控制器

打開car_model.py，並執行。

Week12

Assignment 3

IPv4 Ubuntu對外連線設定

新增一個新的NAT網路。

編輯NAT網路。

設定4個新的連接埠轉送埠號。

設定主機網路為NAT Network。

啟動ubuntu，開啟LXTermial，輸入sudo apt install net-tools，輸入密碼kmol2020。

cd tmp　ls　輸入ping 127.0.0.1測試。

cd cd2020pj1　git pull。

接著輸入sudo vi wsgi.py編輯內容。

將近端的host改為10.0.2.4，接著按Esc並輸入:wq存儲。

開啟leo，並開啟cd2020pj1.leo，點選Nav搜尋allowExt。


點選fileuploadfrom，在第10行新增ttt存儲。

cd..，在tmp下輸入sudo vi oauth_scrum.txt。

按i編輯隨便打，接著按Esc並輸入:wq存儲。

cd cd2020pj1，並輸入python3 wsgi.py啟動。

網址為https:/127.0.0.1:18443/alogin。

帳號及密碼都為admin，進入後點選fileuploadform，點選要上傳的ttt檔。

點選download list檢查檔案是否上傳完成。
